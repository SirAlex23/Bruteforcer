BruteForcer

Proyecto educativo: modos Diccionario y Fuerza Bruta (limitada) para hashes SHA256.

Ver en GitHub ⤓ Descargar ZIP
Tecnologías: Python 3 • hashlib • itertools • uso en laboratorio
Importante: Solo usar en entornos controlados (tu equipo, VMs o laboratorios). Probar contra hosts o cuentas de terceros sin permiso puede ser ilegal.

Resumen

Este proyecto incluye dos modos: Modo 1 — Diccionario (usa "diccionario.txt") y Modo 2 — Fuerza bruta limitada (genera combinaciones hasta una longitud máxima). El objetivo es ilustrar y comparar ambas técnicas.

Cómo probar (local)

  1. Clona el repo: 
    git clone https://github.com/siralex23/bruteforcer.git
cd bruteforcer
  2. Opcional: crea y activa un entorno virtual: 
    python3 -m venv env
source env/bin/activate
  3. Ejecuta el script y elige modo: 
    python3 bruteforcer.py
# Cuando se muestre el menú elige 1 (Diccionario) o 2 (Fuerza bruta)
